# Healthcare Management System

This project is part of the Postman API Hackathon, aiming to build a comprehensive Healthcare Management System that integrates various features to streamline patient care and hospital management. The system utilizes APIs for efficient workflows and data management.

## Features :
* Nearby Hospital Locator
* Tracks and displays nearby hospitals on an interactive map.
* Fetches hospital data based on the userâ€™s current location.
* Displays hospital details, including name, address, contact information and generative summary.

## API Integration:

This project uses multiple external APIs for various functionalities:

* Google Maps API: For locating nearby hospitals and displaying maps.
* Gemeni API : For generating the summary of the located doctors.
